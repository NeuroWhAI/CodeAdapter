# Code Adapter
다른 라이브러리의 인터페이스를 강제시켜서 로직 코드의 변경을 최대한 줄일 수 있게 해주는 라이브러리.

### Project List
  - Code Adapter : 강제시킨 인터페이스가 될 라이브러리.
  - CodeAdapterSFML : SFML 라이브러리의 인터페이스를 강제시킨 예제.
  - TestLab : 테스트 및 데모 프로젝트.

### Features
  - 2D Graphics
  - GUI
  - Scene System
